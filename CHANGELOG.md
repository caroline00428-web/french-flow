- 纯前端SPA，React+Vite+TypeScript+Tailwind CSS
- 智谱AI GLM-4-Flash 提供翻译和语音识别
- 浏览器Web Speech API 提供TTS发音
- IndexedDB (Dexie.js) 存储用户数据
- GitHub Pages 部署，自动构建
